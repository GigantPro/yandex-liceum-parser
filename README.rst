Installation
^^^^^^^^^^^^

**From source** 

*Dependencies*:

* `poetry` (*pip install poetry*)

.. code-block:: console

    git clone https://github.com/GigantPro/yandex-liceum-parser.git; cd yandex-liceum-parser-master
    poetry run install

-------------
